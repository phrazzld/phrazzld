# Phaedrus Raznikov

I build things with AI and ship them.

Solo founder at [Misty Step](https://mistystep.io). One engineer, no employees, fleet of AI agents. The bet: the right tools make headcount irrelevant.

### Developer tools

- **[thinktank](https://github.com/misty-step/thinktank)** - Multi-LLM synthesis CLI. Query multiple models, get one answer.
- **[moonbridge](https://github.com/misty-step/moonbridge)** - MCP bridge for delegating between AI CLIs.
- **[vox](https://github.com/misty-step/vox)** - Invisible macOS dictation. Hotkey to clean text, anywhere.
- **[ponder](https://github.com/phrazzld/ponder)** - Fast note-taking CLI. Rust.
- **[glance](https://github.com/phrazzld/glance)** - Autogen directory summaries. Go.

### Products

- **[Chrondle](https://chrondle.com)** - Daily history game. Guess the year in six tries.
- **[Time is Money](https://github.com/phrazzld/timeismoney)** - Chrome extension. Prices as hours of work.
- **[Bitcoin Price Tag](https://github.com/phrazzld/bitcoin-price-tag)** - Chrome extension. Fiat prices in sats.
